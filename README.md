imageFilter是一个图像处理的类库，用于实现实现图像滤镜与处理，包括明度、饱和度、高斯模糊、做旧效果、四角压暗、灰度、切割等等。

* 版本：1.0
* 作者：茅晓锋(加里)
* 标签：图像处理 canvas html5
